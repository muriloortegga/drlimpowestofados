# Logos do Dr. Limpow

Esta pasta foi criada para armazenar os arquivos vetoriais (SVG) do logotipo da Dr. Limpow.

## Arquivos:

- `logo-original.svg`: O SVG contendo o mascote (a gota com rostinho) nas cores originais da marca.

## Como usar:

Você pode adicionar novas versões do logotipo (como versões horizontais, monocromáticas ou negativas) nesta pasta em formato `.svg`.
Para que o sistema altere as cores dinamicamente via CSS, utilize `currentColor` ou variáveis CSS nos preenchimentos (`fill`) e contornos (`stroke`) dos elementos internos do SVG.
